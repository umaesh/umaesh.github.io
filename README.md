2. **About** - Brief bio and professional summary
3. **Projects** - Showcase of development projects with links to GitHub repos
4. **Skills** - Technical and soft skills organized by category
5. **Contact** - Social media links (GitHub, LinkedIn, Email, Twitter)

## How to Use

Simply visit [https://umaesh.github.io](https://umaesh.github.io) to view the live portfolio.

## Customization

To personalize this portfolio:

1. Update the project cards in `index.html` with your actual projects
2. Modify the skills in the Skills section to match your expertise
3. Update social media links in the Contact section
4. Replace "your.email@example.com" with your actual email
5. Adjust the color scheme in `styles.css` (CSS variables in `:root`)
6. Add your own bio and description in the About section
